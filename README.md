# tasmotaread
Simple tasmota smart-meter command-line tool
see also https://tasmota.github.io/docs/Smart-Meter-Interface/

## Requirements

- curl and jq must be installed.

## Development


Install:
```sh
sudo make install
```

Uninstall:
```sh
sudo make uninstall
```

## Usage
```
tasmotaread <host>
```
